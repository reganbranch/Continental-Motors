User Guide

  1. Start by selecting the plane model that you want the torque values for such as “TSIO-550k”. If you want a general torque values, select “General” instead.

  2. After selecting your engine model, fill out the rest of the required fields on the form.

  3. Make sure to select the button that corresponds to the desired ouput format. Torque values can be displayed in either feet pounds or inch pounds. To switch between the two, select the opposite button and click the “Give Torque Values” button again.

  4. Once all the required fields have been sufficiently filled, click the "Give Torque Values" button to produce the torque value corresponding to your piece.

  5. When you are done, click the "Reset Filters" button to clear all boxes.

*Leverage may affect accuracy of data supplied by application.
